# Flask Stock Market Analysis API

This project implements a Flask-based REST API using Yahoo Finance data.

## APIs Implemented
1. Company Information API (GET)
2. Stock Market Data API (GET)
3. Historical Market Data API (POST)
4. Analytical Insights API (POST)

## How to Run
1. Install dependencies:
   pip install flask yfinance

2. Run the application:
   python app.py

## Testing
GET APIs tested using browser/Postman.
POST APIs tested using Postman.
Screenshots of outputs are included in the output folder.
